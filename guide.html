왜 상세페이지에 영상을?

글과 이미지만으로는 한계가 있어요. 영상이 들어가면 체류시간이 늘고, 제품을 실제로 쓰는 모습을 직접 보여줄 수 있죠. 경쟁 상품과 차별화되는 가장 간단한 방법이에요.

문제는 쿠팡 상세페이지가 HTML 편집 기반이라, 유튜브가 주는 기본 코드를 그대로 넣으면 자동재생이나 반복이 안 된다는 점. 그래서 코드를 살짝 바꿔줘야 해요.

1. 유튜브 준비

1

유튜브 업로드

먼저 유튜브에 영상이 있어야겠죠.

유튜브에 로그인 → 우측 상단 [동영상 업로드] → 영상 등록.

2

퍼가기 허용 설정

유튜브 스튜디오 > [콘텐츠] > 영상 클릭 → [더보기] → '퍼가기 허용' 체크.

중요 이게 꺼져 있으면 쿠팡 상세페이지에서 영상이 안 떠요.

2. 유튜브에서 코드 복사하기

일반 영상인지 쇼츠인지에 따라 방법이 달라요.

1

영상 아래 공유 버튼 누르기

2

팝업에서 퍼가기 </> 선택

3

우측 하단 복사 클릭

iframe 코드가 클립보드에 저장돼요.

쇼츠

영상 위에서 마우스 우클릭 → 소스 코드 복사

쇼츠는 공유 메뉴에 퍼가기가 없어요. 대신 영상을 우클릭하면 "소스 코드 복사" 항목이 떠요.

참고 쇼츠도 일반 영상과 똑같이 처리돼요. 복사한 코드를 그대로 변환기에 붙이면 됩니다.

3. Video Embed에서 변환하기

1

변환기에 붙여넣기

복사한 코드를 변환기 입력칸에 붙여넣으면 제목과 영상 ID가 자동으로 잡혀요.

2

사이즈 고르기

  • 가로 16:9 — 780×439 (쿠팡 기본 너비)
  • 4:3 — 780×585
  • 세로·쇼츠 — 500×889

직접 숫자를 넣어도 돼요. 비율 고정은 기본 켜져 있어요.

3

결과 코드 복사

3번 칸 오른쪽 위 복사 버튼 한 번이면 끝.

4. 쿠팡 윙에 붙여넣기

1

쿠팡 윙 → 상품관리 → 상세설명 편집

2

에디터에서 HTML / 소스 모드로 전환 후 붙여넣기

일반 편집 모드에 붙이면 코드가 텍스트로 들어가요. 꼭 HTML 편집 모드에서 붙여넣어야 영상으로 보입니다.

3

저장 → 미리보기에서 재생 확인

영상이 뜨고 자동으로 재생되면 성공이에요.

팁 & 주의사항

  • 자동재생엔 mute가 필수. 브라우저 정책상 소리 켠 채 자동재생은 막혀 있어요. 변환기는 자동으로 mute 처리해요.
  • 저작권 있는 영상은 본인 것이나 허가받은 것만 써주세요.
  • 모바일 환경에서도 보기 편하게 가로 780 기준을 지키는 걸 추천해요. 쇼츠는 500×889 프리셋.
  • 영상이 너무 크면 상세페이지 로딩이 느려질 수 있어요. 한 페이지에 여러 개 넣는 건 비추.
  • 쿠팡 정책 변경으로 iframe이 막히는 경우가 드물게 있어요. 미리보기에서 꼭 확인해주세요.

자주 묻는 질문

쇼츠랑 일반 영상이 다르게 처리되나요?

아니요. Video Embed 내부에선 똑같이 처리돼요. 영상 ID만 쓰기 때문에 비율만 달라요 (쇼츠는 9:16 세로).

자동재생이 왜 안 되죠?

대부분 mute가 꺼져 있을 때예요. 변환기는 자동으로 mute를 넣어요. 그래도 안 되면 브라우저의 자동재생 차단 설정을 확인해보세요.

iframe이 쿠팡에서 막히진 않나요?

2026년 현재 쿠팡 윙 상세설명 HTML 모드에서 유튜브 iframe은 허용돼요. 정책이 바뀔 수 있으니 공지 확인 권장.

반복 재생이 안 되는데요?

loop 파라미터가 동작하려면 playlist 파라미터에도 같은 영상 ID가 들어가야 해요. 변환기 출력물엔 이미 둘 다 포함돼 있습니다.

영상 ID가 뭐예요?

유튜브 주소의 11자리 짧은 코드예요. 예: z4a_sGDFmAA. 변환기가 자동으로 찾아줍니다.

무료인가요?

네. 로그인도 없어요. 붙여넣고 복사만 하면 끝.
