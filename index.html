어떻게 하나요? 사용법 바로가기 →

1코드 붙여넣기

유튜브 코드를 못 찾았어요.

2사이즈

기본 780. 쿠팡 상세페이지 기준.

3결과 코드

위에 코드를 붙여넣으면 여기에 결과가 떠요.

유튜브 코드 복사하는 법

1. 영상 아래 공유 버튼

유튜브 영상 플레이어 하단의 공유 버튼을 눌러요.

2. 퍼가기 선택

팝업에서 </> 퍼가기를 눌러요.

3. 복사 클릭

iframe 코드가 뜨면 오른쪽 아래 복사를 누르세요.

쇼츠: 영상 우클릭 → 소스 코드 복사

쇼츠는 공유 메뉴에 퍼가기가 없어요. 영상 위에서 마우스 우클릭 → 소스 코드 복사.

안 보이나요? 유튜브 스튜디오 > 콘텐츠 > 영상 > 더보기 > '퍼가기 허용'이 켜져 있어야 해요.
